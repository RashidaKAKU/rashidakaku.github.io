常见问题

.py 文件无法正常运行

一般是由于缺少运行库，可以尝试使用 PIP 解决

pip install -r requirements.txt

Linux 系统缺少 espeak

ArchLinux 以及基于此系统的发行版 Linux 用户可以通过 yay 安装 AUR 包 espeak 解决这个问题

Ubuntu 系统可以通过安装 espeak 解决

如果尚未解决，可以尝试重新安装 pyttsx3 模块
